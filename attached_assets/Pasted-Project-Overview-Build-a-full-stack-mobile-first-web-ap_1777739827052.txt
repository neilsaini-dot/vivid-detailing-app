Project Overview
Build a full-stack, mobile-first web app for Vivid Detailing, a premium automotive appearance shop located at 2-70 Nicholas Lane, Charlottetown, PEI. Phone: 902-267-7775. Email: contact@vividpei.com.
The app is a digital service advisor — it guides customers to the right service by goal, shows accurate dynamic pricing, and converts visitors into booked appointments. It should feel like a premium automotive brand, not a generic booking tool.

Tech Stack (fixed — do not deviate)

Frontend: Next.js 14 (App Router) + Tailwind CSS
Backend: Next.js API routes (no separate Express server)
Database: Supabase (PostgreSQL + Supabase Storage + Supabase Auth)
Auth: Supabase Auth — email magic link only, no passwords
Payments: Stripe (optional, scaffold only in V1)
CRM: GoHighLevel via webhook (V1 — no direct GHL REST API)
Hosting-ready: Vercel-compatible build

Pricing Engine (/lib/pricing.ts)
Build all pricing logic here. Never hardcode pricing in UI components.
Pricing Rules
Each service has a pricing_rule field. Support these rules:
typescripttype PricingRule =
  | 'flat'                        // same price regardless of vehicle
  | 'seasonal_flat'               // flat promo, ignores vehicle type
  | 'fixed_by_vehicle_type'       // explicit price per vehicle in service_prices table
  | 'vehicle_multiplier_round5'   // base × multiplier, round UP to nearest $5
  | 'quote_based'                 // no price shown, CTA only
  | 'admin_configurable'          // admin sets price, show "Pricing Available Upon Request" until set

const VEHICLE_MULTIPLIERS = { car: 1.0, suv: 1.2, truck: 1.3, van: 1.4 }

function roundUpToNearestFive(amount: number): number {
  return Math.ceil(amount / 5) * 5
}

function calculatePrice(service, vehicleType): number | 'quote' | 'pending'

GoHighLevel Integration
V1 uses webhook only — no direct GHL REST API calls.
Store GHL_WEBHOOK_URL in environment variables.
Trigger a POST to GHL_WEBHOOK_URL on:

New booking submitted
PPF/paint correction quote request submitted
Abandoned booking (customer reached Step 6 but did not confirm — fire after 30min)

Webhook payload:
json{
  "event": "booking_created",
  "customer": {
    "name": "string",
    "email": "string",
    "phone": "string"
  },
  "vehicle": {
    "type": "car|suv|truck|van",
    "year": "string",
    "make": "string",
    "model": "string"
  },
  "booking": {
    "service_category": "string",
    "package": "string",
    "addons": ["string"],
    "total_estimate": 0,
    "appointment_at": "ISO8601",
    "notes": "string",
    "is_quote_based": false
  },
  "tags": ["Detail|Summer Special|Ceramic|Tint|Paint Correction|PPF Quote|High Value Lead"],
  "source": "vivid-app"
}
For PPF quote: "event": "ppf_quote_request" + add "coverage_option" field.
For abandoned booking: "event": "booking_abandoned" + last step reached.
